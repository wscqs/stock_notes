## 🌟 核心价值

为股票投资者提供​**​离线化​**​的复盘记录解决方案，通过结构化数据管理与富文本笔记，实现买卖策略的沉淀与回溯。

​**​便捷数据导入导出​**​分享：1.亲朋好友分享自己的数据，支持切换本地多账号。2.备份自己数据导入覆盖。

***

### 🔥 特色功能亮点

#### 📝 智能交易管理

*   核心：满足买卖与临近买卖（±3%阈值可调）
*   计划支持：价格，市值，市盈率
*   左滑“持有/置顶/标签/收藏/删除“操作

#### 📝 多账号数据

*   支持便捷的数据导入导出（可直接分享到微信，微信里面直接导入成功）
*   支持切换本地多账号数据

#### 🗃️ 数据机制

*   双重删除保护（回收站+永久删除）
*   数据导入导出

#### 🔍 高效检索体系

*   搜索
*   状态过滤（全部/持有/收藏/删除）
*   满足买卖/临近买卖
*   标签

#### 🎨 体验优化设计

*   中英双语，日夜间模式
*   多平台小组件入口
*   笔记富文本编辑
*   列表长按批量操作等
*   名言警句功能等
