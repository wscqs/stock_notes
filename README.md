# 股票笔记 - 本地化股票复盘助手 [Flutter 多平台开源]

📱 **纯本地应用 | 隐私安全 | 多平台支持 | 数据导入导出 | 中英双语 | 日夜间模式**

**GitHub 开源地址** <https://github.com/wscqs/stock_notes> | **Releases 下载体验** <https://github.com/wscqs/stock_notes/releases>

Android 也可从蒲公英下载 <https://www.pgyer.com/QBbOMfya>

## 🌟 核心价值

为股票投资者提供**离线化**的复盘记录解决方案。通过结构化数据管理与富文本笔记，实现买卖策略的沉淀与回溯，所有数据本地存储，无需担心隐私泄露。

**便捷数据导入导出**：
1. 支持将数据分享给好友，好友可直接导入学习
2. 支持本地多账号切换，方便管理不同策略组合
3. 支持数据备份与恢复

## 🔥 特色功能亮点

### 📝 智能交易管理

- **买卖条件追踪**：自定义价格 / 市值 / 市盈率的目标值与阈值（±3% 可调），自动计算是否满足买入或卖出条件
- **临近提醒**：实时显示距离目标条件的差距，一目了然
- **快捷操作**：列表左滑支持「持有 / 置顶 / 标签 / 收藏 / 删除」一键操作
- **批量管理**：长按列表进入批量选择模式，效率翻倍

### 📈 股票详情记录

- **实时行情一览**：股票名称、代码、市价、市盈率、市净率、总市值等关键数据集中展示
- **三维买卖计划**：支持按**价格 / 市值 / 市盈率**分别设定买入价与卖出价，实时显示与当前值的差距百分比（如 `买: 1.6%`、`卖: 2.4%`），并为每条计划填写独立理由
- **条件状态追踪**：自动计算并标注当前是否「满足买入」、「满足卖出」或「临近买入」（状态标签实时显示在右侧）
- **持有成本价**：记录持仓成本，辅助盈亏判断
- **备注与事件记录**：独立备注区 + 事件记录区，方便记录公司公告、个人思考等非交易类信息
- **交易记录**：底部「交易记录」模块支持新增每一笔买入 / 卖出 / 备注操作，形成完整交易流水

### 🗃️ 数据导入导出 & 多账号

- 支持 `.db` 文件导入导出，可直接分享到微信，好友微信内直接打开导入
- 支持本地**多账号数据切换**，轻松管理不同组合或策略
- 数据库文件需包含 `stocknotes_` 前缀以被识别

### 🛡️ 双重删除保护

- 删除操作先进入**回收站**，可随时恢复
- 回收站内支持**永久删除**，防止误操作丢失数据

### 🔍 高效检索体系

- **关键词搜索**：快速定位股票或笔记
- **状态过滤**：全部 / 持有 / 收藏 / 删除（回收站）
- **条件过滤**：满足买入 / 满足卖出 / 临近买入 / 临近卖出
- **标签过滤**：自定义标签归类，灵活筛选

### 🎨 体验优化设计

- **中英双语**：支持简体中文 / 英文切换
- **日夜间模式**：适配不同使用场景
- **多平台支持**：已打包 Android / Windows / macOS 安装包
- **名言警句**：随机展示投资名言，保持理性
- **富文本笔记**：基于 `flutter_quill`，支持图文混排
